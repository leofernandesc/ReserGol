# ReserGol
Repositório para armazenar os códigos referentes ao trabalho final da disciplina Modelagem e Projeto de Sistemas, ministrada pela Profa. Danielle Pontes

*Minitutorial para rodar o código localmente*

Para ativar o ambiente virtual: 

No windows: 
.venv\Scripts\activate

No linux:
source .venv/bin/activate

Se houver alguma alteração na pasta requirements, atualizar o venv com o seguinte comando:
pip install -r requirements.txt

Após ativar o ambiente virtual, mude para o diretório Cadastro_e_Login e rode o comando:
python app.py
